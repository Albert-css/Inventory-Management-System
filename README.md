# Inventory Management System

## Описание проекта

**Inventory Management System** - это десктопное приложение для управления товарами на складе, разработанное на Java с использованием JavaFX. Программа предназначена для малого и среднего бизнеса, нуждающегося в эффективном учете товарных запасов.

**Основные функции:**
- Управление товарами (добавление, редактирование, удаление)
- Визуализация статусов товаров через цветовую кодировку
- Фильтрация и сортировка товаров
- Статистика и аналитика склада
- История всех операций с товарами
- Импорт/экспорт данных в CSV формате
- Автоматическая проверка уникальности товаров

**Целевая аудитория:** 
- Владельцы малого бизнеса
- Складские менеджеры
- Ритейлеры
- Логистические компании

## Установка и запуск

### Требования к системе:
- Java JDK 11 или выше
- JavaFX SDK 11 или выше
- Минимум 2 ГБ оперативной памяти
- 100 МБ свободного места на диске

### Пошаговая инструкция:

1. **Установка Java:**
   ```
   # Для Ubuntu/Debian:
   sudo apt install openjdk-11-jdk
   
   # Для Windows:
   # Скачайте и установите JDK с oracle.com
   ```

2. **Установка JavaFX:**
   - Скачайте JavaFX SDK
   - Распакуйте архив в удобное место (например, `C:/javafx-sdk-11.0.2/`)

3. **Запуск приложения:**
   ```
   # Компиляция и запуск через командную строку:
   javac --module-path "путь/к/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml -d out src/**/*.java
   java --module-path "путь/к/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml -cp out application.Main
   ```

4. **Запуск через IDE (IntelliJ IDEA/Eclipse):**
   - Откройте проект в IDE
   - Добавьте JavaFX в classpath
   - Укажите VM options: `--module-path "путь/к/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml`
   - Запустите класс `application.Main`

## Примеры использования

### Добавление товара:
1. Нажмите кнопку "Добавить товар"
2. Заполните поля в диалоговом окне:
   - Имя: "Молоко"
   - Бренд: "Агрокомплекс" 
   - Цена: 89
   - Количество: 100
   - Среднее количество: 150
3. Нажмите "Добавить"

### Фильтрация товаров:
```
// Программный пример фильтрации
filterModule.setMinQuantity(50); // Показать товары с количеством >= 50
filterModule.setShowZeroQuantity(false); // Скрыть товары с нулевым количеством
filterModule.setSearchText("молоко"); // Поиск по имени
```

### Экспорт данных:
1. Выберите в ComboBox "Файл" → "Сохранить"
2. Укажите имя файла (например, `inventory_backup.csv`)
3. Данные будут сохранены в CSV формате

### Импорт данных:
1. Выберите "Файл" → "Загрузить"
2. Выберите CSV файл с данными
3. Данные автоматически загрузятся в систему
