# План тестирования проекта Inventory-Management-System

## 1. Цели тестирования
1.  Функциональная валидация всех требований (Визуал, функционал склада).
2.  Проверка надежности и корректности конфигурации интерфейса.
3.  Оценка удобства работоспособности программы.

## 2. Область покрытия
* **Модули:** Товар, Сортировка, Фильтрация, Контроллер, Пользовательский Интерфейс.
* **Типы:** Функциональное тестирование (FT), Пользовательское тестирование (UT), Тестирование безопасности (Security).

## 3. Среда и Инструменты
* **Среда:** ORACLE java-sdk 24, Intellij IDEA, Windows/Linux.
* **Инструменты:** Intellij IDEA, Unit-тесты.

## 4. Критерии завершения
* Успешное выполнение не менее **85%** Функциональных тест-кейсов.
* Отсутствие **критических** ошибок при тестировании безопасности.
